1-my_list.py Doctest
===============================

Test the MyList class:

>>> MyList = __import__('1-my_list').MyList

Create a MyList instance and append elements:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)

Print the list (should be in original order):
>>> print(my_list)
[1, 4, 2, 3, 5]

Print the sorted list:
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Print the original list again (should remain unchanged):
>>> print(my_list)
[1, 4, 2, 3, 5]

Test with empty list:
>>> empty_list = MyList()
>>> empty_list.print_sorted()
[]

Test with single element:
>>> single = MyList()
>>> single.append(42)
>>> single.print_sorted()
[42]

Test with already sorted list:
>>> sorted_list = MyList([1, 2, 3, 4, 5])
>>> sorted_list.print_sorted()
[1, 2, 3, 4, 5]

Test with reverse sorted list:
>>> reverse_list = MyList([5, 4, 3, 2, 1])
>>> reverse_list.print_sorted()
[1, 2, 3, 4, 5]
